# شاتنا - تطبيق الدردشة العربي

## نظرة عامة
تطبيق دردشة عربي مخصص للتعارف والمحادثة بين المستخدمين من مختلف الدول العربية.

## المميزات
- تسجيل حساب بسيط (اسم مستخدم + صورة + دولة)
- غرف دردشة للدول العربية
- واجهة عربية كاملة بتصميم حديث
- نظام رسائل فورية
- نظام عضويات مميزة بثلاث مستويات (Bronze, Red, Gold)

## الهيكل التقني
- React 18 للواجهة
- TailwindCSS للتصميم
- LocalStorage لحفظ بيانات المستخدم

## الملفات الرئيسية
- `index.html`: الصفحة الرئيسية
- `app.js`: منطق التطبيق الأساسي
- `components/`: مكونات الواجهة
  - Header.js: الهيدر وبيانات المستخدم
  - RegisterModal.js: نافذة التسجيل
  - RoomsList.js: قائمة غرف الدردشة
  - CountryRoom.js: واجهة غرفة الدردشة
  - ChatMessage.js: عرض الرسائل
  - MembershipCard.js: بطاقة عرض العضوية
  - MembershipPlans.js: قائمة خطط العضويات

## التطوير المستقبلي
- إضافة دردشة خاصة بين المستخدمين
- نظام عضويات مميزة
- إضافة المزيد من الغرف
- تحسين نظام الإشعارات
